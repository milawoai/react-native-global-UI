# react-native-global-UI

#### react-native-global-UI provide some UI which can use with API.
#### User can use those UI in everyWhere without declare them in JSX


# Usage
================================================

### 1）import UI

```
import {
  Loading,
  HUD
} from 'react-native-global-ui'
```

### 2）show it！

```
Loading.show()
```

### 3) hide it!

```
Loading.hide()
```
